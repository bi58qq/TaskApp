# Task Tracker App
 - Console-based Task and Project Tracking system
 - Built on C# and the .Net
 - Provide the features to  create, update, search, sort, and track the tasks throughout their life cycle.

The application is based on the principles of object-oriented programming, such as encapsulation, abstraction, etc. 
Service classes are used to implement core functionalities
Data models show tasks and their characteristics like priority, status, assignee, and due date.

The data is stored as tasks which are stored as JSON files so that information is not lost between executions of the program. 
When crucial actions of the system like creating and updating the tasks take place, a log file is updated to facilitate auditing and debugging. 
The system has search and sorting algorithms that will help find tasks by their title as well as sorting them by priority or deadline.
